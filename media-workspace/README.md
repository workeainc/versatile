# Media Workspace

Drop original client/project media here. This folder is intentionally ignored by Git so the repository stays small.

Use this structure:

```text
media-workspace/projects/{project-slug}/originals/
media-workspace/projects/{project-slug}/notes/
```

Example:

```text
media-workspace/projects/gulshan-family-apartment/originals/
media-workspace/projects/gulshan-family-apartment/notes/project-info.txt
```

Do not manually rename, compress, or resize originals before adding them. Keep the highest-quality source files here.

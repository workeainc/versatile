# 13 - Topical Authority Matrix

Purpose: build authority by covering complete topic ecosystems, not isolated keywords. Each cluster should prove experience, expertise, authority, and trust through services, guides, projects, FAQs, images, and local relevance.

## Authority Principles

- Write for entities and relationships, not exact-match keywords alone.
- Add first-hand project details whenever possible.
- Support each service with local pages, case studies, and educational guides.
- Use real photos and project stories as the strongest trust signal.
- Avoid duplicated city pages with only location names swapped.

# 10 - Monthly SEO Checklist

Purpose: keep SEO moving after launch without random changes.

## Technical

- Check sitemap returns 200.
- Check `www` redirects to non-www.
- Check robots.txt references canonical sitemap.
- Check no new broken internal links.
- Check no accidental soft 404s.
- Run production build before deploying.
- Check Search Console indexing warnings.

## Content

- Publish or prepare 2 useful articles.
- Upgrade 1 existing article or service page.
- Add at least 3 internal links to a priority page.
- Review titles/meta for high-impression low-CTR pages.
- Add real project content if photos/details are ready.

## Local SEO

- Add new Google Business Profile photos.
- Publish one GBP post.
- Ask recent clients for genuine reviews.
- Reply to new reviews.
- Confirm NAP consistency across site, GBP, and directories.

## Analytics

- Review organic sessions.
- Review project page views.
- Review service page views.
- Review consultation starts/submissions.
- Review phone/email/WhatsApp clicks.
- Review top landing pages.

## Authority

- Add or update 2-4 legitimate citations or partner mentions.
- Look for supplier, contractor, client, or publication link opportunities.
- Avoid low-quality link packages.

## Image SEO

- Process any new project images.
- Confirm filenames, alt text, captions, and dimensions.
- Confirm hero images are not oversized.
- Add project image references to structured data when ready.

## Monthly Output

At the end of each month, write:

- What changed.
- What improved.
- What declined.
- What to publish next.
- What technical issue needs attention.
- What business information is still missing.

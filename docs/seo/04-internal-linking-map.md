# 04 - Internal Linking Map

Purpose: help visitors and search engines move from educational content to service pages, project proof, local relevance, and consultation.

## Core Link Hubs

| Hub | Supporting pages that should link to it |
| --- | --- |
| `/` | Major service pages, local pages, footer, GBP profile, citations. |
| `/projects` | Homepage, service pages, project articles, case study CTAs. |
| `/services` | Homepage, local pages, blog posts, footer. |
| `/request-consultation` | Service pages, project pages, blog posts, local pages, contact page. |
| `/interior-design-dhaka` | Blog posts with broad Dhaka context, service pages, footer or local section. |

## Blog Publishing Rule

Every new article should include:

- 1 link to a service page.
- 1 link to a project page if relevant.
- 1 link to a local landing page if location intent exists.
- 1 link to a related article when available.
- 1 conversion path to `/request-consultation` when commercially appropriate.

## Anchor Text Rules

Use descriptive anchors:

- Good: `restaurant interior design service`
- Good: `office interior design in Dhaka`
- Good: `Courtyard House residential project`
- Avoid: `click here`
- Avoid: repeating the same exact anchor unnaturally on every page.

## Monthly Check

Each month, pick one priority page and add 3-5 helpful internal links to it from relevant existing content.

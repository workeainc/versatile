## Monthly Decisions

Use the dashboard to decide:

- Which article should be expanded.
- Which local page needs stronger project proof.
- Which service page should receive more internal links.
- Which project case study should be upgraded.
- Which CTA placements are producing qualified leads.

Do not add features unless the dashboard suggests a real business reason.
